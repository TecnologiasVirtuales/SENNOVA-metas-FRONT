# Proyecto metas (front)
Este proyecto fue construido usando angular 18
## Requisitos
1. Node 22 o superior
## Instalación
1. Clonar el repositorio : git clone https://github.com/TecnologiasVirtuales/SENNOVA-metas-FRONT.git
2. Instalar las dependencias : npm install
## Ejecución
1. Iniciar el proyecto : ng serve
